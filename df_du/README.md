# df & du - Command

# Disk free

### Show disk free-space & available-space & total-space
```
df
```

### show disk free-space & available-space & total-space (in megabyte format)
```
df -m
```

### show disk free-space & available-space & total-space (in gigabyte format)
```
df -H 
df -h
```

du
